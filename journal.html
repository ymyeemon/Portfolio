Web Application Development

Yee Mon's Travel Journal

Overview

This web application is created with features to create, read, update and delete travel journals effortlessly. Utilizing Express.js, EJS, and Bootstrap, it supports full CRUD operations for documenting and sharing travel experiences as a personal blog or journals.

Key Features

Utilize EJS templating engine to inject dynamic data into HTML templates, allowing for real-time updates to the content displayed on the page based on user interactions and database changes.

Use CRUE Operations such as Adding new journal entries(C), View and retrieve existing entries from the database(R), Edit and Update existing entries(U) and Remove or Delete journal entries as needed(D).

Create (C)

Create operation is a feature whereby new journal entries can be added with their respective Location, Travel Period, Title of the journal, Text, Date of the entry and last but not least, an image from the device.

Read (R)

Upon reaching at the main landing page or home page, all the existing journal entries with their specific detailed information and images can be seen within each card.

Update (U)

Update operation provides a feature to freely update each journal information such as changing Texts, Title, Travel period, Date and even Images.

Delete (D)

Delete operation provide a functionality to remove or delete any journal entries. Upon clicking the delete button of each journal, the alert message will appears at the top to validate the choice again.
