Travel List App

Overview

Travel List App is created by using React framework to helps users efficiently organize their packing process. The app allows users to add items to a list with their respective quantities, mark items as packed, and track their packing progress. It ensures an organized packing experience, keeping everything in check.

Key Features

Add Items with Quantity: Users can input an item name and the quantity they plan to pack, which will be added to the list dynamically.

Mark as Packed: Individual items can be marked as packed, making it easy to track progress.

Packing Progress Tracking: A status bar at the bottom displays the total number of items in the list, the number of packed items, and the packing completion percentage in real-time.

Removing All Items: A single click clears the entire list, allowing for quick resets.

Adding Items to the List

Users can key in the items they want to pack along with the quantities and each item will be added to the packing list accordingly. After adding the items, a status bar at the bottom will display how many items are currently in the list and the number of packed items with a completion percentage as well.

Checking Items as Packed

Users can click the checkbox beside each item in the list to mark it as packed. Once marked, the status bar at the bottom dynamically updates to reflect the changes, showing the number of packed items and the percentage of completion.
